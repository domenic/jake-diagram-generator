What's all this?

A "Jake diagram", named after their inventor, the imitable Jake Archibald, depicts the joint session history of a web browser.

Each numbered column represents a step in the session history. Each row represents a navigable, as it transitions between different URLs and documents. The URLs are given by the text content of the cells, and the documents are given by the background color of the cells. A given navigable might not exist at a given step, in which case the corresponding cells are empty.

This tool allows you to generate and edit Jake diagrams, using a hacked-together domain-specific language. It's intended to be used mainly by contributors to the HTML Standard, especially those working on the navigation and session history rewrite.
